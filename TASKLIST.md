# GenGate — OpenCode Tasklist

## Current objective
Bootstrap the repository architecture for a Locket-like private social app with messaging, moments, location, and profile.

## Rules for implementation
- Keep scope at MVP scaffolding first
- Prefer clean monorepo structure over premature feature completeness
- Do not fake production completeness
- Leave clear TODO markers where product decisions are pending
- Keep commits/task boundaries small and understandable

## Immediate tasks
1. Initialize monorepo structure for apps and packages
2. Set up shared TypeScript/tooling config
3. Bootstrap backend app with health endpoint and domain modules
4. Define initial database schema for auth/profile/friends/moments/messages/locations
5. Bootstrap web app routes and shell
6. Bootstrap mobile app routes and shell
7. Bootstrap desktop app shell
8. Add environment examples and startup docs
9. Summarize what runs and what is still stubbed

## Required report format after each batch
1. % progress
2. files/folders created or changed
3. what is runnable now
4. blockers / decisions needed
5. next implementation batch
